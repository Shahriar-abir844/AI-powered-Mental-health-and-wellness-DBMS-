About this project

This repository contains a minimal static demo for a DBMS-backed AI-powered mental health and wellness project. The dashboard shows sample DB statistics, engagement/mood trends and a placeholder AI mood analyzer.

Intended usage: local preview, design iteration, and demonstrating where integrations (database, model API) should be connected. No production clinical inference is performed here.
